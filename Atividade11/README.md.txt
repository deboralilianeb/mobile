Atividade XI 
-Construa uma aplicação que contenha uma
interface gráfica que solicite ao usuário que
tente adivinhar um número sorteado entre 1 e
10.
-O palpite deve ser digitado em uma caixa de
texto (EditText) logo abaixo da mensagem
acima.
-Após a digitação e o pressionamento de um
botão para conferência, sua aplicação deve
informar se o usuário acertou ou errou.
-Utilize uma activity para entrada de dados e
outra para exibição do resultado.
