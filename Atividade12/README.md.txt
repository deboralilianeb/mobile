Atividade XII
Objetivo: tratar múltiplos eventos disparados pela interação do usuário com a interface gráfica do aplicativo.

Construa uma calculadora simplificada que permita que o usuário informe um número real em um campo de entrada e outro número real em outro campo de entrada.

Após, o usuário pode clicar em botões que representam as quatro operações aritméticas fundamentais (soma, subtração, multiplicação e divisão) e um rótulo deve exibir o resultado desejado.

O layout deve ser o apresentado a seguir.

|_________________________| // Campo de entrada 1
|_________________________| // Campo de entrada 2
| + | | - | | * | | / | // Botões para seleção da operação
|_________________________| // Rótulo para exibição do resultado
