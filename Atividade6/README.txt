Atividade VI
-Construa uma aplicação que contenha duas
activities.
-A Activity inicial deve conter um botão que
permita invocar a segunda.
-A segunda Activity deve devolver um valor
aleatório como retorno.
-A Activity inicial deve exibir o valor sorteado em
seu botão e informar que novo sorteio é
possível.
-Todo o ciclo de vida deve ser monitorado.
