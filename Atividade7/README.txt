Atividade VIII
-Construa uma aplicação que contenha duas
activities.
-A Activity inicial deve conter um botão que
permita invocar a segunda.
-Esta Activity deve gerar três números
aleatórios que serão passados para a
segunda.
-O primeiro número deve ser um valor de 0 a 3, que
indica uma operação aritmética a ser processada 
(0= soma, 1 = subtração, 2 = multiplicação e 3 =divisão)
 e os demais números serão os operandos.
