Atividade IX
-Construa uma aplicação que contenha quatro
activities.
-A Activity inicial deve escolher, aleatoriamente,
qual das outras três invocar após o usuário
clicar em sua tela.
-Cada uma das três activities deve conter um
vetor de strings com 10 elementos quaisquer e
um desses elementos deve ser selecionado
aleatoriamente e exibido em sua Activity.
-Após clicar no texto que exibe o elemento
sorteado, a Activity deve devolver o resultado
para a Activity inicial, que exibirá o resultado.
-Todas as strings sorteadas devem ser
monitoradas através do log.
